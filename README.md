# Simple Interest Calculator

This project calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Features
- Accepts principal amount
- Accepts interest rate
- Accepts time period
- Calculates simple interest

## Technologies Used
- Python / JavaScript / HTML (choose what you used)

## Example
Input:
Principal = 1000
Rate = 5
Time = 2

Output:
Simple Interest = 100
